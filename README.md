# BMI Company Profile (WIP)

Website company profile untuk **PT. Bina Mitra Internusa (BMI)**, dibangun dengan Next.js (App Router) dan dilengkapi area admin/CMS untuk mengelola konten (mis. berita, galeri, layanan, karir, dan halaman marketing lainnya).

## Status

**Masih dalam tahap pengembangan (Work in Progress).**

## Tujuan Publikasi

Repository ini dipublikasikan **untuk portofolio**. Lihat bagian lisensi di bawah untuk batasan penggunaan kode.

## Menjalankan Secara Lokal

```bash
npm install
npm run dev
```

Buka `http://localhost:3000`.

## Struktur & Dokumentasi

- Dokumentasi internal tersedia di folder `DOCS/`.
- Halaman marketing ada di `app/(marketing)/`.
- Area admin ada di `app/admin/`.
